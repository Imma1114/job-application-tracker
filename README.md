# Job Application Tracker

A simple web app to **track and manage your job applications**.  
Add jobs, see their status, and view statistics at a glance. Supports **dark mode** and **persistent storage**.

---

## Features

- Add new job applications (Job Title, Company, Status, Date)  
- Track status (Applied, Interview, Rejected, Offer)  
- Search and filter jobs  
- Delete applications  
- Dark mode / Light mode toggle  
- Interactive chart with application status statistics  

---

## Tech Stack

- HTML5  
- CSS3  
- JavaScript (ES6)  
- Chart.js  

---

## How to Use

1. Open `index.html` in your browser  
2. Add a job using the form  
3. View stats and chart updates automatically  
4. Search, filter, or delete applications as needed  
5. Toggle between dark and light mode
